# **App Name**: Praxis Online

## Core Features:

- Doctor Profiles: Display information about each doctor in the practice.
- Contact Form: Enable patients to send inquiries via a contact form.
- AI-Powered Symptom Checker: Allow patients to check their symptoms and get a possible diagnosis with the help of an AI tool.

## Style Guidelines:

- Primary color: Calming light blue (#7EC4CF), reflecting trust and health, like praxiszentrum-im-ring.ch.
- Background color: Very light blue (#F0F8FF), almost white to ensure clarity, similar to praxiszentrum-im-ring.ch.
- Accent color: Soft teal (#66B2B2), provides a modern, friendly feel and ensures good contrast, inspired by praxiszentrum-im-ring.ch.
- Body and headline font: 'Montserrat' (sans-serif), for a modern and readable look, aligning with the professional aesthetic of praxiszentrum-im-ring.ch.
- Use simple and clean icons representing services and information, like the existing 'praxiszentrum im ring'.
- Clean, spacious layout with clear sections for easy navigation, similar to 'praxiszentrum im ring'.
- Subtle transitions and animations to enhance user experience without being distracting.
