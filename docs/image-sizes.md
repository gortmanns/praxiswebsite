# Bildgrössen-Referenz für die Webseite

Diese Datei dient als Referenz für die optimalen Bildgrössen, die auf der Webseite verwendet werden. Die Bilder sollten vor dem Hochladen auf diese empfohlenen Maximalgrössen komprimiert werden, um die Ladezeiten zu optimieren, ohne an Qualität zu verlieren.

## Globale Elemente

### 1. Logo im Header

-   **Datei:** `public/images/praxiszentrum-logo.png`
-   **Anzeigeort:** Haupt-Header auf allen Seiten.
-   **Empfohlene max. Auflösung:** `800px × 162px`
-   **Hinweis:** Das Logo wird auf grossen Bildschirmen mit einer maximalen Breite von 781px angezeigt, behält aber sein Seitenverhältnis bei. 800px Breite bietet einen guten Puffer.

### 2. Logo im Footer

-   **Datei:** `public/images/mehrfacharzt-logo.png`
-   **Anzeigeort:** Footer, Sektion "Zertifizierte Qualität".
-   **Empfohlene max. Auflösung:** `300px × 96px`
-   **Hinweis:** Das Bild ist relativ klein. Eine Breite von 300px ist mehr als ausreichend.

## Startseite (`/`)

### 1. Hero-Bild

-   **Datei:** `public/images/luftbild.jpg`
-   **Anzeigeort:** Grosses Hauptbild auf der Startseite.
-   **Empfohlene max. Auflösung:** `1920px × 915px`
-   **Hinweis:** Das Bild füllt die gesamte Breite des Bildschirms. 1920px ist eine sichere maximale Breite für grosse Desktop-Monitore.

### 2. Kooperationspartner-Logos

-   **Dateien:** z.B. `public/images/VASC-Alliance-Logo.png`, `go-medical-logo.png`, etc.
-   **Anzeigeort:** Im Abschnitt "Kooperationspartner" auf der Startseite.
-   **Empfohlene max. Auflösung:** `600px × 300px` (für ein Seitenverhältnis von 2:1)
-   **Hinweis:** Die Karten haben ein 2:1-Verhältnis. Eine Breite von 600px ist ein guter Richtwert, um auf allen Bildschirmgrössen scharf zu bleiben.

## Team-Seite (`/team`)

### 1. Porträtbilder (Ärzte & Personal)

-   **Dateien:** z.B. `public/images/team/Ortmanns.jpg`, `Garcia.jpg`, etc.
-   **Anzeigeort:** Auf der Team-Seite für Ärzte und Praxispersonal.
-   **Empfohlene max. Auflösung:** `400px × 600px`
-   **Hinweis:** Die Bilder haben ein festes Seitenverhältnis von 2:3 (Breite zu Höhe).

### 2. Logo/Positions-Bilder (innerhalb der Karten)

-   **Dateien:** z.B. Logos von `orthozentrum-bern.ch`, etc.
-   **Anzeigeort:** Innerhalb der Ärzte- und Dienstleisterkarten.
-   **Empfohlene max. Auflösung:** `400px × 100px`
-   **Hinweis:** Diese Logos sind in der Höhe begrenzt. Eine maximale Höhe von 100px ist ein guter Richtwert.

## Leistungs-Seite (`/leistungen`)

### 1. Leistungskarten-Bilder

-   **Dateien:** z.B. `public/images/leistungen/audiometrie.jpg`, `ekg.jpg`, etc.
-   **Anzeigeort:** Die Kacheln auf der Leistungsseite.
-   **Empfohlene max. Auflösung:** `600px × 600px`
-   **Hinweis:** Die Bilder sind quadratisch. 600px ist eine sichere Grösse für eine scharfe Darstellung im 3-spaltigen Layout.

## Notfall-Seite (`/notfall`)

### 1. Ambulanz-Bild

-   **Datei:** `public/images/rtw-bern.jpg`
-   **Anzeigeort:** Neben der Notrufnummer 144.
-   **Empfohlene max. Auflösung:** `600px × 400px`

### 2. Medphone & Tox Info Logos

-   **Dateien:** `public/images/medphone_logo.png`, `toxinfo-logo.svg`
-   **Anzeigeort:** In den jeweiligen Notfall-Sektionen.
-   **Empfohlene max. Auflösung:** `800px` Breite (SVG ist verlustfrei skalierbar)
-   **Hinweis:** Eine grosse Ausgangsbreite für PNGs ist hier gut, da sie auf grossen Bildschirmen viel Platz einnehmen.
